ovofans — Albums API

A small Express API serving Drake albums (for your client to fetch).

API Endpoints

  • GET /api — API index (JSON)
  • GET /api/albums — Returns all albums (JSON)
  • GET /api/albums/:id — Returns a single album by id
  • POST /api/upload — Upload image (form-data with field image)

View All Albums (JSON)

Place album cover images in public/images/. Example cover paths: /images/scorpion.jpg
